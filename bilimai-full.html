📌 Тақырып мақсаты

Квадрат теңдеулерді шешудің барлық тәсілдерін үйрену: дискриминант формуласы, Виет теоремасы, толық квадратқа шығару.

📐 Негізгі формула

ax² + bx + c = 0
D = b² − 4ac
x = (−b ± √D) / 2a

🧠 Виет теоремасы

Егер x₁ және x₂ — теңдеудің түбірлері болса:

x₁ + x₂ = −b/a
x₁ · x₂ = c/a

💡 Есте сақта

▸ D > 0 → екі нақты түбір
▸ D = 0 → бір түбір (x = −b/2a)
▸ D < 0 → нақты түбір жоқ

01:30

x² − 5x + 6 = 0 теңдеуінің түбірлерін тап.

A
x₁=1, x₂=6
B
x₁=2, x₂=3
C
x₁=−2, x₂=−3
D
x₁=1, x₂=−6
✓ Дұрыс! D = 25−24 = 1. x = (5±1)/2 → x₁=3, x₂=2. Немесе Виет: 2+3=5, 2·3=6. ✔
